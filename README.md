# Pomodoro Timer (Tkinter GUI)

A simple Pomodoro Timer application built with Python and Tkinter.

This project was built using functions, global variables, Tkinter widgets, event handling, and countdown logic.

---

## Features

* Start a Pomodoro work session
* Automatic switching between work sessions and breaks
* Short break after each work session
* Long break after every four completed work sessions
* Visual check marks for completed work sessions
* Reset button to stop and restart the timer
* Protection against starting multiple timers simultaneously

---

## What I practiced

* Tkinter GUI development
* Canvas widget and image handling
* Button commands and event-driven programming
* Working with global variables
* Using `window.after()` for countdown functionality
* Canceling scheduled events with `after_cancel()`
* Updating Canvas items dynamically with `itemconfig()`
* Basic Pomodoro timer logic
* Managing application state
